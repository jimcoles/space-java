Contains all the original hard-coded language elements for ROML including non-terminals and terminals.

Package Specification

Eventually we will have a version of this package that is itself generated from a ROML model. Many element classes are named per the following:
xyzDefn - the named defining expression
xyzExpr - the defining expression
xyz - the runtime instance that adheres to am xyzDefn

This applies to Transform, Equation, Action, and Type.
